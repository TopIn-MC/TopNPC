# TopNPC
Top NPC for TopIn
## Install
- You can either install as an plugin
- Or install it in the `addon` folder of TopIn
## Setup
1. Select an npc
2. Type `/settopnpc <datalist> <index>`
3. The top npc is set
## Remove
1. Select an npc
2. Type `/removetopnpc`
3. It's now removed
## Permissions
* `topin.npc` for the commands
